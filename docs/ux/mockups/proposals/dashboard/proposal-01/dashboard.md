# Dashboard - Proposal 01
![Preview](./dashboard.PNG)

### Idea
Dashboard enfocado primero en el diseño movil, contiene algunos elementos de gamefication para hacerlo un poco mas entretenido, condensa mas o menos todo lo que puedes hacer con la app.

### Decisiones
- Lista vertical de partidos
- Ver predicciones anteriores
- Balance de puntuación
- Ranking semanal reducido y a la vista
- Acceso rápido a todo

### Pros
- Va al grano
- Rápido de usar
- Se centra en lo mas importante, que es predecir

### Contras
- Podria estar cargado de redundancia
- Escala regular si son muchos partidos
- Diseño un poco corporativo

### Mejoras
- Quizas una barra de progreso
- Tal vez subdivir las cosas de alguna forma en semanas
- Tal vez recurrir a un buscador o algun filtro
- Remover la tasa de acierto y racha, o simplificar
